Quick Start & Usage

Add a Business

Navigate to Add Business, complete the form and submit. reCAPTCHA v3 protects submissions.

Interact & Review

Visit the Home page to view listings and recommendations. Clicking "View Reviews" records view/click analytics which improve recommendations.

Admin Analytics

Judges can review recorded analytics at Admin Analytics. This page is restricted to authorized admin users.

Accessibility & Tips
  • Keyboard accessible navigation via standard links.
  • Forms include server-side validation; error messages display on the page.
  • If reCAPTCHA blocks you during local testing, set test keys or run in dev mode.
For Judges

Use the Admin Analytics page to evaluate recommendation behavior. To allow a user to view this page locally, set the environment variable ADMIN_USERS to a comma-separated list of usernames (or add an is_admin column to the users table).
